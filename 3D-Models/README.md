# 3D 打印模型文件

本目录包含 HeadNeckControl 仿生颈部控制系统的 3D 打印模型文件。

## 目录结构

```
3D-Models/
├── OBJ/          # Wavefront OBJ 格式模型
├── 3MF/          # Bambu Studio 切片格式文件
└── README.md      # 本文件
```

## 打印建议

### 所有主体
- **材料**：PLA 或 PETG（推荐 PETG）
- **打印参数**：参考3mf文件里的配置


### 颈部连接件
- **材料**：PA（尼龙）
- **打印参数**：参考3mf文件里的配置
- **打磨**：拆完支撑后最好把接触面打磨一下

## 注意事项

**因为制作模型单位问题，如果自己把模型导入切片软件需要放大十倍！！！放大十倍！！！**
