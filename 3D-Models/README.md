# 3D 打印模型文件

本目录包含 HeadNeckControl 仿生颈部控制系统的 3D 打印模型文件。

## 目录结构

```
3D-Models/
├── OBJ/          # Wavefront OBJ 格式模型
├── 3MF/          # 3D Manufacturing Format 格式模型
├── STL/          # STL 格式模型
└── README.md      # 本文件
```

## 文件格式说明

### OBJ 格式（推荐）
- **格式**：Wavefront OBJ
- **优点**：
  - 通用的 3D 模型交换格式
  - 支持多边形、法线和纹理坐标
  - 大多数 3D 打印切片软件支持
- **适用场景**：通用 3D 打印、模型编辑

### 3MF 格式
- **格式**：3D Manufacturing Format
- **优点**：
  - 微软开发的开放标准
  - 支持颜色、材质和纹理
  - 文件体积小，压缩效率高
  - Windows 10/11 原生支持
- **适用场景**：Windows 用户、需要颜色打印

### STL 格式
- **格式**：Stereolithography
- **优点**：
  - 最常见的 3D 打印格式
  - 所有切片软件都支持
  - 简单稳定
- **缺点**：
  - 不支持颜色和纹理
  - 不包含模型元数据
- **适用场景**：单色 3D 打印

## 打印建议

### 舵机支架
- **材料**：PLA 或 PETG（推荐 PETG）
- **填充率**：20-30%
- **层高**：0.2mm
- **支撑**：根据模型结构决定
- **打印温度**：
  - PLA: 200-210°C
  - PETG: 230-240°C

### 颈部连接件
- **材料**：PETG 或 ABS（需要更高的强度）
- **填充率**：30-40%
- **层高**：0.2mm
- **打印温度**：
  - PETG: 230-240°C
  - ABS: 240-250°C（需要加热打印床）

### 外壳和装饰件
- **材料**：PLA（可选颜色）
- **填充率**：15-20%
- **层高**：0.2mm
- **打印温度**：200-210°C

## 后处理

1. **去除支撑**：小心移除打印过程中生成的支撑
2. **打磨**：用砂纸（400-800 目）打磨表面
3. **组装**：
   - 确保所有孔位对齐
   - 使用 M3 螺丝固定
   - 测试舵机安装是否顺畅
4. **上色**（可选）：
   - 使用丙烯颜料
   - 喷涂前先打磨并清洁表面

## 打印软件推荐

- **Cura**：免费，功能强大
- **PrusaSlicer**：开源，专为 Prusa 打印机优化

## 注意事项

1. **打印前检查**：
   - 确认模型尺寸与你的舵机匹配
   - 检查模型是否有缺陷
   - 预览切片结果

2. **首次打印**：
   - 建议先打印小部件测试
   - 验证尺寸是否正确
   - 确认打印质量可接受

3. **材料选择**：
   - PETG：更好的韧性和耐温性
   - PLA：更容易打印，适合静态部件
   - ABS：更强的耐温性，但需要加热床

## 故障排除

**问题：部件无法组装**
- 检查打印尺寸是否正确
- 确认填充率足够（建议 20-40%）
- 打印时是否翘曲

**问题：舵机安装不顺畅**
- 检查孔径是否足够
- 可能需要轻微扩孔
- 检查模型是否打印歪斜

**问题：模型强度不足**
- 增加填充率
- 使用 PETG 或 ABS
- 检查层高是否过大
